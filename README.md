# codealpha_HR-analytics_Dashboard
The Human Resources Analytics Dashboard is an interactive Power BI dashboard that helps analyze workforce data. It provides insights into employee distribution, attrition, recruitment, salary, satisfaction, and performance. The dashboard enables HR teams to make informed decisions using visual analytics.

## Project Objective
1.Monitor employee information across departments.

2.Analyze employee attrition.

3.Evaluate employee satisfaction and performance.

4.Study recruitment sources.

5.Support HR decision-making through interactive visualizations.

 ## Dataset Used
The dataset was created in Microsoft Excel and contains employee information.


## Business Questions (KPIs)
1.How many employees are currently working?

2.What is the employee attrition rate?

3.What is the average employee salary?

4.What is the average employee age?

5.What is the average satisfaction score?

6.What is the average performance rating?

7.Which department has the highest number of employees?

8.Which department has the highest attrition?

9.Which recruitment source hired the most employees?

10.Which age group has the highest employee count?

## Process
1.Created an HR dataset in Microsoft Excel.

2.Imported the dataset into Power BI Desktop.

3.Cleaned and transformed the data using Power Query.

4.Created DAX measures for HR KPIs.

5.Designed interactive visualizations.

6.Added slicers for Department, Gender, and Year.

7.Built a professional dashboard with HR insights.


## Dashboard Features
KPI Cards (Employees, Attrition Rate, Average Salary, Average Age, Satisfaction, Performance)
Employees by Department (Bar Chart)
Gender Distribution (Donut Chart)
Attrition by Department (Bar Chart)
Recruitment Source (Donut Chart)
Performance by Department (Line Chart)
Satisfaction Score Distribution (Column Chart)
Employee Age Group Analysis (Column Chart)
Key Insights Panel
Department, Gender, and Year Slicers
